# Changelog

All notable changes to EditPDFree Utils will be documented in this file.

For web-based PDF editing, visit [EditPDFree.com](https://www.editpdfree.com)

## [1.0.0] - 2026-02-16

### Added
- Initial release of editpdfree-utils
- `getPdfInfo()` - Extract PDF metadata and information
- `mergePdfs()` - Combine multiple PDF files
- `splitPdf()` - Split PDF into separate pages
- `extractPages()` - Extract specific pages from PDF
- `getPageCount()` - Get number of pages in PDF
- `rotatePdf()` - Rotate PDF pages
- Full TypeScript support with type definitions
- Comprehensive README with examples
- MIT License

### Features
- Works with both file paths and Buffers
- Zero configuration required
- Lightweight with minimal dependencies
- Browser and Node.js compatible APIs
